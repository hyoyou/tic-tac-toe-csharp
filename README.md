# Tic Tac Toe [![TravisCI](https://travis-ci.com/hyoyou/tic-tac-toe-csharp.svg?branch=master)]

## Prerequisites
* C#
* NUnit


## Setup
* Install dependencies with:

```
$ nuget restore TicTacToe.sln
```


## Running the Tests
* To run unit tests:

```
$ nunit-console ./Tests/bin/Debug/Tests.dll
```
